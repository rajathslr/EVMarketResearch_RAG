**Introduction to Site Layout Creation and Editing in the Mapper App**
- This video tutorial demonstrates how to create and edit a **site layout** using the Mapper app.
- The **site layout** serves as a virtual map of a photovoltaic (PV) site and is used in the monitoring platform for **remote diagnostics**.
- In the **site list**, there are two categories of sites:
- **On-device sites**: Sites opened and saved locally on the device.
- **Online sites**: Sites saved on the server, accessible remotely.
- To begin creating a site layout, users need to:
- Tap on the relevant site.
- The layout screen will initially be empty.
- Tap the **plus (+) button** to start adding system components.

**Adding System Components to the Site Layout**
- Users can add components by tapping:
- **Inverter**:
- Add a single inverter or multiple inverters simultaneously.
- When adding multiple inverters, specify:
- The **number of inverters**.
- Their **orientation**.
- After selection, tap **Done**.
- **Module group**:
- Add one or more module groups according to the site’s design.
- For each module group, specify:
- Number of **modules per column** and **per row**.
- Module **orientation**: Landscape or vertical.
- Module **tilt** and **azimuth** angles.
- Tap **Done** when finished.
- If the PV site comprises multiple facets or several module groups, continue adding module groups accordingly.
- Positioning modules:
- Drag and drop module groups to the correct location on the layout.

**Editing and Managing Layout Elements**
- To **delete an element** (e.g., inverter or module group):
- Long press the element and drag it to the trash bin icon.
- To **edit module groups**:
- Double tap on the group.
- Change parameters like **tilt**, **azimuth**, and **orientation**.
- Tap **Done** to save changes.
- For sites with **obstacles**:
- Use the **hide/show** feature.
- Tap the edit icon, select modules to hide, then tap **Done**.
- To unhide modules, repeat the process selecting modules to unhide.
- An **undo/redo** function is available for convenience during editing.

**Layout Display and Customization Options**
- Use the **fit to screen icon** to scale the entire layout to fit the device screen for better visualization.
- Change the layout’s background through the **settings icon**:
- Switch between **dark mode** and **light mode**.
- Change the background to **satellite view** to assist in precise navigation and placement of components on the site.
- Optionally, show **roof edges** if previously drawn in the **SolarEdge Designer** software, allowing clearer outline of the site roof.
- **All changes made in the layout are automatically saved as a draft** in the monitoring platform.

**Summary and Saving**
- When the site layout visually matches the physical installation, the user is done with the editing process.
- The draft state means edits are saved online automatically without requiring manual saving actions.
- The video emphasizes smooth integration between the Mapper app and the SolarEdge monitoring platform for **efficient remote site management**.

### **Highlighted Terminology**
- **PV site**: Photovoltaic solar power installation.
- **Inverter**: Device converting DC electricity from solar panels to AC electricity.
- **Module group**: Grouped solar modules arranged in specific rows and columns.
- **Tilt**: Angle of solar modules relative to the horizontal plane.
- **Azimuth**: Compass direction the modules face relative to true north.
- **SolarEdge Designer**: Software used to draft site outlines and roof edges used in the layout.
- **Monitoring platform**: The online system that visualizes and diagnoses PV sites remotely.
